# 🚀 OSANTIX - Социальная сеть нового поколения с полным шифрованием
## OSANTIX - это не просто мессенджер. Это полноценная социальная экосистема, где ваша приватность и безопасность стоят на первом месте. Проект, созданный с душой NatanOsant и Eqwerik, объединяет в себе все современные технологии общения: от надёжных чатов до встроенных игровых механик.

# ✨ Основные возможности
## 🔒 Безопасность и приватность (E2EE)
Сквозное шифрование (End-to-End Encryption): Все сообщения, файлы и данные в группах защищены по протоколу ECDH (P-256) + AES-GCM. Даже мы не можем прочитать вашу переписку.

Верификация ключей: Сверяйте уникальные "безопасные коды" с собеседником, чтобы убедиться в отсутствии MITM-атак.

Блокировка приложения: Защитите доступ к своим чатам с помощью PIN-кода с автоматической блокировкой при бездействии.

Режим невидимки: Контролируйте свой онлайн-статус.

## 💬 Мессенджер и чаты
Личные и групповые чаты: Общайтесь один на один или создавайте группы с приглашениями по ссылке.

Расширенные сообщения: Редактирование, удаление, пересылка, закреплённые сообщения и «Избранное».

Реакции и ответы: Выражайте эмоции с помощью реакций и цитируйте сообщения в ответах.

Голосовые сообщения: Записывайте и отправляйте голосовые заметки.

Поиск: Глобальный поиск по всем чатам, включая зашифрованные.

## 🎮 Игровой центр (Геймификация)
Уровни и XP: Получайте опыт за активность в чатах.

Валюта "Pro100 Coin": Зарабатывайте редкую валюту, которую можно тратить в магазине.

Магазин значков: Продавайте и покупайте эксклюзивные значки у других пользователей.

Достижения и задания: Выполняйте ежедневные квесты (например, написать 10 сообщений или заглянуть в топ) и получайте награды.

Рулетка: Испытайте удачу за игровую валюту.

Рейтинг (Топ): Соревнуйтесь с друзьями по уровню и количеству монет.

## 📢 Каналы и группы
Публичные каналы: Создавайте каналы для трансляции новостей, где писать могут только владельцы.

Группы с приглашениями: Создавайте открытые и закрытые группы. Управляйте заявками на вступление.

## 📞 Звонки
Аудио и Видео-звонки: Встроенные WebRTC-звонки с поддержкой TURN-серверов для работы за NAT и фаерволами.

Демонстрация экрана: Делитесь своим экраном во время звонка.

## 🎨 Интерфейс и персонализация
Современный дизайн: Стильный интерфейс в стиле, вдохновлённом лучшими мессенджерами.

Темы оформления: Светлая, тёмная, высокий контраст и авто-переключение по расписанию.

Настраиваемый акцент: Выберите любимый цвет интерфейса.

Ресайз сайдбара: Изменяйте ширину боковой панели под себя.

## ⚙️ Ключевые технологии
Frontend: HTML5, CSS3 (с кастомными свойствами), JavaScript (ES6+).

Backend & Database: Firebase (Authentication, Cloud Firestore, Storage).

Real-time: Firestore Realtime listeners для мгновенного обновления сообщений.

WebRTC: Для аудио/видео звонков.

Security: Web Crypto API для реализации сквозного шифрования.
